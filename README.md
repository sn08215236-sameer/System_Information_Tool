# System_Information_Tool

## Project Description
This is a Python-based System Monitoring application developed as part of the **Operating Systems** course. The tool provides a graphical user interface (GUI) to monitor real-time hardware statistics and system configurations.

## Features
* **OS Information:** Displays the current operating system and release version.
* **CPU Monitoring:** Shows the number of CPU cores and real-time usage.
* **Memory Tracking:** Provides details on total and available RAM.
* **Network Details:** Displays the System Hostname and local IP Address.

## Project Documentation
The repository is organized into phases to reflect the development lifecycle:
1. **Source Code (`main.py`)**: The primary Python script utilizing `Tkinter` for the GUI and `psutil` for system metrics.
2. **Phase 1**: Contains the initial implementation details and project scope.
3. **Phase 2**: Includes the **Component Diagrams** and architectural logic of the system.
